Validation Checklist

Use this sequence during the lab viva or final demo.

  1. Start ryu-manager src/controller/static_routing_controller.py.
  2. Launch sudo python3 src/topology/static_topology.py.
  3. Run pingall and verify host-to-host connectivity.
  4. Dump flows from s1, s2, and s3.
  5. Reinstall the controller rules and rerun python src/tools/validate_routes.py --compare-snapshot.
